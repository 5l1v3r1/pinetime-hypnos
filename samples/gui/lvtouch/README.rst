.. _touchscreen-sample:

Touchscreen Basic Sample
########################

Overview
********

This sample application displays touchscreen-values x and y in the center of the screen.

Requirements
************

Pinetime watch
zephyr cst816s driver

Building and Running
********************

west build -p -b pinetime samples/gui/lvtouch
